How To Use This Template:
========================
Wordpress Admin:
----------------
1. Plugins : Install and activate free plugins
------------

    1.Elementor
	2.Elementor Addon Elements
	3.Flexible Elementor Panel
    4.Header, Footer & Blocks for Elementor
    5.Page scroll to id

2. Templates : 
--------------
    Import Templates > Choose File > Import Now [***import file is a json file]

3. Pages : 
----------
    1.Add New > Add Title > Create New Page with a name for your project
    2.Edit With Elementor

4. Elementor Editor :
--------------------- 
    1.Click on File Symbol to "Add Template"
    2.Go to "My Template"
    3.Select the Uploaded JSON file and Insert it
    4.Settings > Page Layout > Elementor Canvas
    5.Publish

5. Settings(Wordpress Admin):
----------------------------
    1.Select Reading
    2.A static Page > Homepage > (Select Your New Created Page.
    3.Save Changes
